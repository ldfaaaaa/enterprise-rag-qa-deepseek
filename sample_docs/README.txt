一、公司简介
星光科技有限公司成立于2015年，总部位于上海，是一家专注于企业数字化转型的高新技术企业。公司旗下拥有云端ERP系统、智能客服平台、数据分析中台三大核心产品线，服务客户超过2000家，覆盖制造、零售、医疗等多个行业。

二、核心产品介绍
1. 星光云ERP：支持财务、采购、库存、销售全流程管理，提供SaaS与私有化部署两种模式，支持多组织、多币种、多语言。
2. 智能客服平台：基于大语言模型构建，支持多渠道接入（网页、企业微信、钉钉），7×24小时自动响应，意图识别准确率高达95%。
3. 数据分析中台：提供实时数据看板、多维报表、预测分析功能，支持对接MySQL、Oracle、Hive等主流数据源。
